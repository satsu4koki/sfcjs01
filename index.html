address要素

HTML ＜address＞要素は、祖先要素の内、直近の ＜article＞ 要素か ＜body＞ 要素の著作者の連絡先情報を表します。後者の場合は、ドキュメント全体に適用します。

例

<address> You can contact author at <a href="http://www.somedomain.com/contact">www.somedomain.com</a>.<br> If you see any bugs, please <a href="mailto:webmaster@somedomain.com">contact webmaster</a>.<br> </address>

p要素

HTML ＜p＞ 要素 (HTML パラグラフ要素) は、パラグラフ (段落) を表します。パラグラフはブロックレベル要素です。

例

<p>This is the first paragraph of text. これは最初のテキストの段落です。 This is the first paragraph of text. これは最初のテキストの段落です。 </p>

*

アスタリスク（*）ですべての要素にスタイルを適用することができます。

例

p * { color:red; }

p要素の中にある要素すべてに適用されます。

.x

クラス名を指定した要素にスタイルが適用されます。

例

.sample{ font-size:18px; color:red; }

#x

基本的には先ほどのクラスと一緒です。idの場合は1つの要素にしか適用されません。

例

#sample{ font-size:16px; }

idはjavascriptやページ間の移動の際にも使われます。
